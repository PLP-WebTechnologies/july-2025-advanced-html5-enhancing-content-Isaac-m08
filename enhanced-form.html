HTML5 Input Types Summary

A quick look at common HTML5 input types and their purposes.
Input Type Description
text Used for plain single-line text input.
email Ensures input matches email format (e.g., user@example.com).
password Masks the input for sensitive information.
date Lets users pick a date from a calendar interface.
number Accepts only numeric input with optional min/max limits.
